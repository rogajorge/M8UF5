HTML

HTML són les sigles Hipper Text Markup Language, que podria traduir-se com llenguatge de marques o marcat d'hipertetx.

El llenguatge HTML s'utilitza per la creacio de pàpinas web. És molt senzill de veure en el codi HTMLd'una pàgina web, l'opcio exacta canvia d'un navegador a un altre i també pot canviar d'una versió a un altra en un mateix navegador, però solen tenir una opció similar.

HTLM està format d'etiquetes que s'escriuen entre els símbols menos què i major què.

Història d'HTML

Els principis del lenguatge HTML venen de l'any 1991, quan Tim Berners-Lee va crear la primera pàgina web

Versions d'HTML

Tim Berners-Lee va definir la primera versió d'HTML a l'any 1991.

A l'actualitat, la darrera versió d'HTML ès HTML5.
